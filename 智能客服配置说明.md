# 智能客服功能配置说明

## 功能概述

智能客服功能已实现，支持调用大模型API进行智能问答。目前支持以下大模型：
- 文心一言（百度）
- 千帆（百度智能云 Qianfan V2）
- 通义千问（阿里）
- OpenAI ChatGPT

## 配置步骤

### 1. 数据库配置

执行 `schema.sql` 文件中的客服相关表结构：
- `tb_customer_service_session` - 客服会话表
- `tb_customer_service_message` - 客服消息表

### 2. 后端配置

在 `application.yml` 中配置AI API信息：

```yaml
# AI API配置
ai:
  api:
    url: # 大模型API地址
    key: # API密钥（建议使用环境变量/密钥管理，不要明文写入仓库）
    type: qianfan # qianfan(千帆V2), wenxin(文心一言), tongyi(通义千问), openai(OpenAI兼容)
    model: # 模型名称，例如：ernie-bot-turbo, gpt-3.5-turbo
```

#### 千帆（Qianfan）V2配置示例（对应文档：`https://cloud.baidu.com/doc/qianfan-api/s/ym9chdsy5`）：
```yaml
ai:
  api:
    url: https://qianfan.baidubce.com/v2/chat/completions
    key: ${QIANFAN_API_KEY:}
    type: qianfan
    model: ernie-3.5-8k
```

#### 文心一言配置示例：
```yaml
ai:
  api:
    url: https://aip.baidubce.com/rpc/2.0/ai_custom/v1/wenxinworkshop/chat/ernie-bot-turbo
    key: your_api_key_here
    type: wenxin
    model: ernie-bot-turbo
```

#### OpenAI配置示例：
```yaml
ai:
  api:
    url: https://api.openai.com/v1/chat/completions
    key: sk-your-api-key-here
    type: openai
    model: gpt-3.5-turbo
```

### 3. 测试模式

如果不配置API，系统会使用模拟回复模式，可以用于测试基本功能。

## API接口说明

### 1. 创建会话
```
POST /api/customer-service/session/create
Headers: Authorization: Bearer {token}
Response: { sessionId, sessionNo }
```

### 2. 获取消息列表
```
GET /api/customer-service/session/{sessionId}/messages
Response: [消息列表]
```

### 3. 发送消息
```
POST /api/customer-service/message/send
Body: { sessionId, content }
Response: { message: AI回复消息 }
```

### 4. 结束会话
```
POST /api/customer-service/session/{sessionId}/end
```

## 前端使用

1. 访问 `/customer-service` 页面
2. 系统自动创建会话
3. 输入问题并发送
4. AI会自动回复

## 注意事项

1. **API密钥安全**：请妥善保管API密钥，不要提交到代码仓库
2. **API成本**：调用大模型API会产生费用，请注意控制调用频率
3. **响应时间**：API调用可能需要几秒钟，请耐心等待
4. **错误处理**：如果API调用失败，系统会返回模拟回复
